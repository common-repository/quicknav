== Description ==
Quicknav a plugin for off-canvas features of WordPress. This plugin allows you to show a menu, WooCommerce product, blog, custom link and much more in your website off-canvas those can quickly and easily navigate by a user.


<h4>Main Features of Quicknav:</h4>
<ul>
 	<li>Menu show in off canvas</li>
 	<li>Menu Select options</li>
 	<li>Custom quick nav create options</li>
 	<li>Woocommerce Product show in off canvas</li>
 	<li>Feature product, Recent product, 5 star product show options</li>
 	<li>Blog post show in off canvas</li>
 	<li>Recent blog, blog category select options</li>
 	<li>Off canvas position change options</li>
	<li>Compatibility with wordpress 5.4+</li>
</ul>

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "quicknav". Click on Install and then Activate the Plugin.

That's it, Quicknav Plugin is now activated on your site!

**The hard way..**

Download "Quicknav WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
